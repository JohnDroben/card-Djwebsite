## Особенности

- **Современный дизайн**: Чистый интерфейс с анимациями
- **Полная адаптивность**: Оптимальное отображение на всех устройствах
- **Разделы сайта**: Главная, Обо мне, Портфолио, Контакты, Новости
- **Интерактивные элементы**: Раскрывающиеся карточки, hover-эффекты
- **Система новостей/блога**: Управление контентом через админку
- **Оптимизированная производительность**: Быстрая загрузка страниц
- **SEO-оптимизация**: Правильные мета-теги и семантическая вёрстка

## Технологии

- **Бэкенд**: Django 5.2
- **Фронтенд**: HTML5, CSS3 (Flexbox, Grid), JavaScript
- **База данных**: SQLite (разработка), PostgreSQL (продакшн)
- **Стили**: CSS-переменные для единого оформления
- **Иконки**: Font Awesome 6
- **Деплой**: Готовая конфигурация для Docker

## Установка

### Предварительные требования
- Python 3.10+
- pip

### Инструкция по установке

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-профиль/portfolio-site.git
cd portfolio-site
```

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/MacOS
venv\Scripts\activate     # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Примените миграции базы данных:
```bash
python manage.py migrate
```

5. Создайте суперпользователя (для доступа в админку):
```bash
python manage.py createsuperuser
```

6. Запустите сервер разработки:
```bash
python manage.py runserver
```

7. Откройте сайт по адресу:
```
http://localhost:8000
```

8. Админ-панель доступна по адресу:
```
http://localhost:8000/admin
```

## Деплой в продакшн

### Настройка продакшн
1. Установите переменные окружения:
```bash
export SECRET_KEY='ваш-секретный-ключ'
export DEBUG=False
export ALLOWED_HOSTS='вашдомен.ru'
```

2. Установите зависимости для продакшна:
```bash
pip install gunicorn psycopg2-binary
```

3. Соберите статические файлы:
```bash
python manage.py collectstatic
```

4. Настройте веб-сервер (пример для Nginx):
```nginx
server {
    listen 80;
    server_name вашдомен.ru;

    location /static/ {
        alias /путь/к/статическим/файлам;
    }

    location / {
        proxy_pass http://localhost:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### Деплой с Docker
1. Соберите Docker-образ:
```bash
docker build -t portfolio-site .
```

2. Запустите контейнер:
```bash
docker run -d -p 8000:8000 --name portfolio portfolio-site
```
